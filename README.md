# Tugas Pemrograman Mobile: Aplikasi Kampusku (CRUD SQLite)

Proyek ini adalah aplikasi pengelolaan data mahasiswa yang mengimplementasikan operasi CRUD (Create, Read, Update, Delete) menggunakan database lokal SQLite.

## Deskripsi Tugas
Aplikasi ini memungkinkan pengguna untuk melakukan manajemen data mahasiswa dengan field penyimpanan sebagai berikut:
* Nomor (NIM)
* Nama
* Tanggal Lahir
* Jenis Kelamin
* Alamat

## Fitur Utama
* Input Data: Menambahkan data mahasiswa baru ke dalam database.
* Ubah Data: Mengedit informasi mahasiswa yang sudah tersimpan.
* Hapus Data: Menghapus data mahasiswa dari database.
* Tampil Data: Melihat daftar seluruh mahasiswa yang terdaftar.

## Informasi Database (SQLite)
Aplikasi ini menggunakan SQLite untuk penyimpanan lokal. 
* Lokasi Penyimpanan: Database disimpan pada direktori eksternal di dalam folder aplikasi, yaitu di `Android/data/com.example.kampusku`.
* Cara Akses Manual: File database ini dapat diakses melalui File Manager, kemudian dapat disalin ke PC untuk dibuka dan dikelola menggunakan aplikasi DB Browser for SQLite.

## Cara Menjalankan
1. Clone atau unduh repositori ini.
2. Buka proyek menggunakan Android Studio.
3. Sinkronkan Gradle dan jalankan di Emulator atau Device fisik.
4. Pastikan aplikasi memiliki izin penyimpanan (Storage Permission) untuk mengakses direktori `Android/data/`.
